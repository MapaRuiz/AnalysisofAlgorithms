Multiplicación de Matrices

Visualización paso a paso del algoritmo naive de multiplicación de matrices

¿Cómo funciona?

Para multiplicar dos matrices A y B, cada elemento C[i][j] de la matriz resultante se calcula como la suma del producto de los elementos correspondientes de la fila i de A y la columna j de B.

Complejidad: Tiempo O(n³), Espacio O(n²)

Fórmula: C[i][j] = Σ(A[i][k] × B[k][j]) para k = 0 hasta n-1

Pasos del Algoritmo:

  1. Para cada fila i de la matriz A
  2. Para cada columna j de la matriz B
  3. Calcular C[i][j] = Σ(A[i][k] × B[k][j]) para todos los k
  4. Repetir hasta completar toda la matriz resultado

Condición: El número de columnas de A debe ser igual al número de filas de B.
