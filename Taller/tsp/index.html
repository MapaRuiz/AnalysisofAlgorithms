Problema del Viajante de Comercio (TSP)

Visualización del algoritmo de fuerza bruta para encontrar la ruta más corta

¿Cómo funciona?

El algoritmo evalúa todas las permutaciones posibles de ciudades para encontrar la ruta más corta que visite todas las ciudades exactamente una vez y regrese al punto de origen.

Complejidad: Tiempo O(n!), Espacio O(n)

Nota: Debido a la complejidad factorial, se recomienda usar máximo 8 ciudades para una visualización fluida.

Permutaciones evaluadas: 0
Distancia actual: -
Mejor distancia: -

Pasos del Algoritmo:

  1. Generar todas las permutaciones posibles de ciudades
  2. Para cada permutación, calcular la distancia total de la ruta
  3. Mantener registro de la ruta con menor distancia
  4. Retornar la ruta óptima encontrada

Fórmula de distancia: √[(x₂-x₁)² + (y₂-y₁)²]
