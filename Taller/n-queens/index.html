Problema de las N-Reinas

Visualización del algoritmo de backtracking para colocar N reinas sin ataques

¿Cómo funciona?

El algoritmo coloca reinas fila por fila en un tablero de ajedrez NxN, asegurándose de que ninguna reina pueda atacar a otra. Utiliza backtracking para retroceder cuando una colocación no es válida.

Complejidad: Tiempo O(N!), Espacio O(N)

Restricciones: Las reinas no pueden atacarse en la misma fila, columna o diagonal.

Intentos: 0
Retrocesos: 0
Soluciones: 0

Pasos del Algoritmo:

  1. Comenzar con la primera fila
  2. Para cada columna en la fila actual, verificar si es seguro colocar una reina
  3. Si es seguro, colocar la reina y avanzar a la siguiente fila
  4. Si no es seguro en ninguna columna, retroceder (backtrack) a la fila anterior
  5. Repetir hasta encontrar una solución o agotar todas las posibilidades

Condiciones de seguridad: No hay otras reinas en la misma columna, diagonal principal o diagonal secundaria.
