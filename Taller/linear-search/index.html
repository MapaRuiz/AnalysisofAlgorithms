Búsqueda Lineal

Visualización paso a paso del algoritmo de búsqueda secuencial

¿Cómo funciona?

La búsqueda lineal recorre secuencialmente cada elemento de la lista hasta encontrar el elemento objetivo o agotar todos los elementos. Es el algoritmo de búsqueda más simple pero no el más eficiente.

Complejidad: Tiempo O(n), Espacio O(1)

Comparaciones: 0

Pasos del Algoritmo:

  1. Comenzar desde el primer elemento del arreglo
  2. Comparar el elemento actual con el objetivo
  3. Si coinciden, devolver el índice
  4. Si no coinciden, avanzar al siguiente elemento
  5. Repetir hasta encontrar el elemento o llegar al final
